Guides to Get Started



Data Provider Getting Started

Bizo Data Providers are paid a share of the revenues generated from any targeted Bizo Ads that utilize their data for bizographic targeting.

Download our "Getting Started With Bizo" quick start guide which contains all of the information you need to become activated as a Bizo Data Provider including:

  • Program Elements
  • Tags and Pixels
  • Privacy Policy
  • Frequently Asked Questions
  • Helpful Links and Contacts





Inventory Provider Getting Started

Bizo Inventory Providers are paid a share of the revenues generated for any targeted Bizo ads that are delivered on their site(s).

Download our "Getting Started With Bizo" quick start guide which contains all of the information you need to become activated as a Bizo Data Provider including:

  • Program Elements
  • Tags and Pixels
  • Pricing Structure
  • Frequently Asked Questions
  • Helpful Links and Contacts
